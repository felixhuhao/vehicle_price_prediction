# 二手车价格预测 - 数据字段说明

数据来源：`used_car_train.csv` / `used_car_testA.csv` / `used_car_testB.csv`

- 测试集：30 个字段（无 `price`）
- 训练集：31 个字段（含预测目标 `price`）

字段分为基础信息、匿名特征和目标字段三大类。

## 基础信息字段

| 字段名               | 含义                                           | 数据类型 | 示例     |
|----------------------|------------------------------------------------|----------|----------|
| `SaleID`             | 交易ID，唯一标识一条交易记录                   | int      | 150000   |
| `name`               | 汽车交易名称（编码后的汽车编码）               | int      | 66932    |
| `regDate`            | 汽车注册日期，格式 YYYYMMDD                    | int      | 20111212 |
| `model`              | 车型编码                                       | float    | 222.0    |
| `brand`              | 汽车品牌编码                                   | int      | 4        |
| `bodyType`           | 车身类型                                       | float    | 5.0      |
| `fuelType`           | 燃油类型                                       | float    | 1.0      |
| `gearbox`            | 变速箱类型（0-自动/1-手动）                    | float    | 1.0      |
| `power`              | 汽车发动机功率                                 | int      | 313      |
| `kilometer`          | 汽车行驶里程（万公里）                         | float    | 15.0     |
| `notRepairedDamage`  | 汽车是否有未修复的损坏（0-否/1-是/`-`缺失）    | str      | 0.0      |
| `regionCode`         | 地区编码                                       | int      | 1440     |
| `seller`             | 销售方                                         | int      | 0        |
| `offerType`          | 报价类型                                       | int      | 0        |
| `creatDate`          | 上架日期，格式 YYYYMMDD                        | int      | 20160329 |
